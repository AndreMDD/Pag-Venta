Tienda demo - BloomCare

Descripción
- Pequeña página web estática para vender productos femeninos.
- Funcionalidades: catálogo, carrito, registro/inicio de sesión (simulado con localStorage), y proceso de pago de demostración.

Cómo probar
1. Abrir el archivo `index.html` en tu navegador (doble clic o arrastrar al navegador).
2. Navega por los productos y haz clic en "Agregar" para añadir al carrito.
3. Haz clic en "Iniciar sesión / Registro" para crear una cuenta o entrar.
4. Pulsa "Carrito" y luego "Pagar" (es un flujo simulado; no hay integración real de pagos).

Notas y siguientes pasos recomendados
- Actualmente la autenticación y el almacenamiento del carrito usan localStorage (solo para demostración). Para producción, integrar un back-end seguro con base de datos y pasarela de pago (Stripe, PayPal, etc.).
- Añadir validación de formularios más robusta y manejo de errores en el servidor.
- Añadir imágenes locales y optimización de rendimiento.
